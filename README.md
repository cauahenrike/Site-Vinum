
 🍷 Vinum — Inteligência que Flui ()

Vinum é um projeto fictício de landing page premium criado com HTML, CSS e JavaScript puro, desenvolvido com foco em design moderno, animações suaves e experiência visual elegante.

O projeto simula uma plataforma de inteligência e análise de dados voltada para empresas, utilizando uma identidade inspirada em vinho, sofisticação e tecnologia.

---

## ✨ Sobre o projeto

A proposta da Vinum é apresentar uma interface moderna de uma plataforma SaaS fictícia, com:

- Visual premium e minimalista
- Animações interativas
- Design responsivo
- Elementos inspirados em dashboards modernos
- Experiência fluida e elegante

O site foi criado com auxílio de Inteligência Artificial para explorar conceitos de UI/UX avançados e front-end moderno.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript Vanilla
- Google Fonts

---

## 🎨 Recursos do site

- Cursor customizado
- Efeitos de glow e blur
- Scroll reveal animations
- Navbar dinâmica
- Carrossel de depoimentos
- FAQ interativo
- Pricing toggle mensal/anual
- Layout totalmente responsivo
- Design inspirado em plataformas SaaS premium

Este projeto foi desenvolvido para:

- estudar experiências modernas de landing pages
